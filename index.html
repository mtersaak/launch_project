Welcome

For our Web Development class we were given "20% time". This was where we were given time at the end of class to explore something we want and to be able to teach someone about it. The final exam is based around building the website around the Bootstrap framework. This website is about a game that I enjoy but can also teach you a thing or two. This game is called Kerbal Space Program.

KSP is a game about building and launching rockets and spacecraft. This game is known for being very accurate to real life in the fact that everything in KSP is 1:10 the scale of real life. This game teaches basic rocket science, orbital mechanics, and aerospace engineering all while keeping you glued to the screen. I used Adobe Premiere Pro to edit my videos, I recorded them using Windows Game Recording on my PC and, of course, used Kerbal Space Program for footage.
